OpenAPI

可用接口与参数说明（中文字段已统一使用英文参数名）

GET /api/nodes
返回所有节点（含 IP 信息与状态）
参数：无
curl -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/nodes
POST /api/auth
登录获取 token（如果未设置密码则无需调用）
Body：{"password":"你的密码"}
curl -X POST -H "Content-Type: application/json" -d '{"password":"your_password"}' http://127.0.0.1:9090/api/auth
GET /api/settings
获取运行设置（external_ip / probe_target / skip_cert_verify）
参数：无
curl -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/settings
PUT /api/settings
更新运行设置（保存到 config.yaml）
Body：{"external_ip":"","probe_target":"www.apple.com:80","skip_cert_verify":true}
curl -X PUT -H "Authorization: Bearer <token>" -H "Content-Type: application/json" -d '{"external_ip":"","probe_target":"www.apple.com:80","skip_cert_verify":true}' http://127.0.0.1:9090/api/settings
GET /api/export
导出可用代理池节点（纯文本，每行一个）
参数：无
curl -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/export
GET /api/export/filter
按条件筛选并返回代理地址（JSON）
参数：
shared_min / shared_max（共享人数范围）
country（国家/地区，支持英文）
ip_src（原生/广播）
ip_attr（住宅/机房）
fraud_max（Fraud Score ≤）
pure_max（纯净度 ≤）
latency_max（延迟 ≤，ms）
curl -H "Authorization: Bearer <token>" "http://127.0.0.1:9090/api/export/filter?shared_max=10&country=United%20States&ip_src=原生&fraud_max=50&pure_max=50&latency_max=500"
POST /api/nodes/{tag}/probe
手动探测单个节点
参数：路径参数 tag
curl -X POST -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/nodes/<tag>/probe
POST /api/nodes/{tag}/release
解除节点拉黑
参数：路径参数 tag
curl -X POST -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/nodes/<tag>/release
POST /api/nodes/probe-all
批量探测节点（SSE 进度流）
参数：
interval_ms（起始检测间隔）
interval_step_ms（每次成功后增加的间隔）
max_interval_ms（最大间隔）
curl -N -X POST -H "Authorization: Bearer <token>" "http://127.0.0.1:9090/api/nodes/probe-all?interval_ms=200&interval_step_ms=100&max_interval_ms=2000"
GET /api/debug
调试信息与探测日志
参数：无
curl -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/debug
GET /api/nodes/config
读取配置节点列表
参数：无
curl -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/nodes/config
POST /api/nodes/config
新增配置节点
Body：{"name":"","uri":"","port":0,"username":"","password":""}
curl -X POST -H "Authorization: Bearer <token>" -H "Content-Type: application/json" -d '{"name":"node","uri":"vless://..."}' http://127.0.0.1:9090/api/nodes/config
PUT /api/nodes/config/{name}
更新配置节点
Body：{"name":"","uri":"","port":0,"username":"","password":""}
curl -X PUT -H "Authorization: Bearer <token>" -H "Content-Type: application/json" -d '{"name":"node","uri":"vless://..."}' http://127.0.0.1:9090/api/nodes/config/<name>
DELETE /api/nodes/config/{name}
删除配置节点
参数：路径参数 name
curl -X DELETE -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/nodes/config/<name>
POST /api/reload
重载配置（中断现有连接）
参数：无
curl -X POST -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/reload
GET /api/subscription/status
查看订阅刷新状态
参数：无
curl -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/subscription/status
POST /api/subscription/refresh
立即刷新订阅
参数：无
curl -X POST -H "Authorization: Bearer <token>" http://127.0.0.1:9090/api/subscription/refresh
